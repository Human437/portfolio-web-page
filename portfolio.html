Calculator Project

This project was inspired by The Odin Project. This is a basic calcualtor app that incorporates PEMDAS. In addition to the basic functions of a calculator, it has a backspace button, all clear button, and previous answer button. The project is nothing crazy, but it taught me the basics of combining HTML, CSS, JS.

Etch a Sketch Project

This project was inspired by The Odin Project. The goal was to replicate an Etch a Sketch board by using HTML, CSS, and JS. This was a great project to learn about the basics of dom manipulation and the basics of event listeners.
